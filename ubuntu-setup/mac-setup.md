# Install software on Mac using homebrew

## Install homebrew

## Install software

1. Install git
brew install git

2. Install tig
brew install tig

3. Install tree
brew install tree

4. Install ag
brew install the_silver_searcher

5. Install ripgrep
brew install ripgrep

6. Install fd to find files
brew install fd

7. Install bat to view files in color
brew install bat

8. Install jq to filter json
brew install jq

9. Install unzip (may not be needed)

10. Install vifm
brew install vifm

11. Install pandoc
brew install pandoc

12. Install lazygit
brew install lazygit

13. Install wuzz http tool (not available in homebrew)

14. Install ctags
brew install ctags

15. Install osquery (query os details using sql - not homebrew

16. Install tmux
brew install tmux
